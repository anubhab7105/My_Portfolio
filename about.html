about me

Security-first builder

I'm Anubhab Das, a first-year B.Sc Cyber Security student at Guru Nanak Institute of Technology (MAKAUT), Kolkata. My journey into tech started with a fascination for how things break — and more importantly, how to make sure they don't.

I specialize in building secure, production-grade web applications that balance great user experience with rock-solid security architecture. From implementing JWT authentication and bcrypt password hashing to preventing SQL injection, XSS, and CSRF attacks, I approach every project with the OWASP Top 10 in mind.

Beyond security, I'm a full-stack web developer comfortable across the entire stack — crafting responsive frontends with HTML/CSS/JS and building backend APIs with Python and Flask. I've completed three internships — at Prodigy InfoTech, iStudio, and Infotact Solutions — where I worked on real-world client projects.

When I'm not writing code, I'm researching the latest vulnerabilities, exploring threat intelligence tools, or sharpening my Python skills. I'm actively looking for opportunities in web development, cybersecurity analysis, or security engineering.
